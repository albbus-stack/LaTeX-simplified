A faster syntax parser to LaTeX.

# Build it

###### Requirements

* A LaTeX compiler (like [latexmk](https://mg.readthedocs.io/latexmk.html))
* Python 3.8
* Install all the requirements in [requirements.txt](requirements.txt).

###### Steps

1. Run `python app.py` in your venv.
